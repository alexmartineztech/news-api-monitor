# News API Monitor

## 🚀 Overview
A Python-based automation tool that fetches real-time news from an external API (NewsAPI) and displays the latest articles based on user input.

This project demonstrates API integration, environment configuration, and data processing in Python.

---

## 🧰 Tech Stack
- Python
- Requests
- REST APIs
- dotenv (environment variables)

---

## ⚙️ Features
- Fetch real-time news from NewsAPI
- Search by custom topic (AI, technology, bitcoin, etc.)
- Display formatted results in terminal
- Secure API key management using environment variables
